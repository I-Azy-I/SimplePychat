Pour que le programme marche, il faudra d'abord installer un/des modules:
- Ouvrez un Terminal
- Déplacez-vous à l'aide de la fonction cd jusqu'au dossier contenant SimplePychat
- Entrez cette commande dans le Terminal:
	
	[sous Linux] $ sudo pip3 install -r requirements
	[sous Windows] pip install -r requirements.txt

Pour lancer le programme, déplacez-vous dans le terminal jusqu'au dossier SimplePychat et entrez simplement:
      python3 Simple_Pychat_Local
